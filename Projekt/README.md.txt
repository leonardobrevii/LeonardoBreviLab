FitnessTracker
Opis projektu

FitnessTracker to aplikacja internetowa stworzona w technologii ASP.NET Core MVC słuzy do organizowania i zarzadzania treningami fitness.

Aplikacja umożliwia dodawanie, edytowanie, usuwanie oraz wyszukiwanie treningów. Dodatkowo użytkownik może zarządzać trenerami i przypisywać ich do poszczególnych treningów.

Funkcjonalności
Dodawanie treningów
Edycja treningów
Usuwanie treningów
Wyszukiwanie treningów
Zarządzanie trenerami
Relacje pomiędzy treningami i trenerami
Walidacja formularzy
Obsługa bazy danych SQLite
Konfiguracja Docker
Technologie
ASP.NET Core MVC
Entity Framework Core
SQLite
Bootstrap
Docker
